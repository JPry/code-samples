## Description ##

Creates custom metaboxes for posts.

## Installation ##

### Manual Installation ###

1. Upload the entire `/jpry-post-metabox` directory to the `/wp-content/plugins/` directory.
2. Activate JPry Post Metabox through the 'Plugins' menu in WordPress.
